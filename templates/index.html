Description of the game

This project is a simple quiz game about the Czech Republic. There are 5 trivia questions about the Czech Republic, and you have to get 3 of them correctly in order to win. You win by simply choosing a part of the flag to uncover, and then answering the question correctly, which will uncover the flag. When the entire flag is uncovered, you win the game, and a won game is added to your profile statistics.
